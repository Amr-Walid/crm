# Project Rules

- **Automatic Commits**: After making any changes or edits to files in this project, you must automatically create a Git commit with a clear, concise commit message describing the changes made. Do not wait for the user to ask for a commit.

- **Review Files Before Answering**: Always view and inspect the contents of target files before responding to the user's questions or request changes, ensuring the answer is fully accurate and tailored to the actual implementation.
